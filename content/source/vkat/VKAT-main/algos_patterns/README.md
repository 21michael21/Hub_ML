# Livecoding Prep — 12h Plan

Все файлы запускаются через `python3 <файл>`. Каждый содержит теорию, примеры, задачи и тесты.

## Порядок работы

| Файл | Блок | Время | Запустить |
|------|------|-------|-----------|
| `01_big_o.py` | Big O time + space | 1.5h | `python3 01_big_o.py` |
| `02_python_builtins_complexity.py` | Python built-ins complexity | 30m | `python3 02_python_builtins_complexity.py` |
| `03_hashmap_pattern.py` | Hash Map паттерн | 45m | `python3 03_hashmap_pattern.py` |
| `04_two_pointers_pattern.py` | Two Pointers паттерн | 45m | `python3 04_two_pointers_pattern.py` |
| `05_sliding_window_pattern.py` | Sliding Window паттерн | 45m | `python3 05_sliding_window_pattern.py` |
| `06_stack_pattern.py` | Stack паттерн | 30m | `python3 06_stack_pattern.py` |
| `07_recursion_pattern.py` | Рекурсия | 45m | `python3 07_recursion_pattern.py` |
| `08_mock_interview.py` | Mock interview | 2h | `python3 08_mock_interview.py` |
| `09_technique_cheatsheet.py` | Техника livecoding | 30m | `python3 09_technique_cheatsheet.py` |

## Как работать с каждым файлом

1. **Запусти** файл — прочитай теорию и вывод
2. **Закрой решения** в коде — попробуй написать сам
3. **Скажи вслух** сложность каждого решения
4. **Сверь** с тестами и комментариями
